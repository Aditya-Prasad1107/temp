# Editing Entries

This guide explains how to edit existing metadata entries in the SDD Automation Tool.

---

## Accessing the Edit Function

Each entry in the metadata table has action buttons in the last column. To edit an entry, you will use these buttons.

## Types of Edits

You can make three types of edits to an entry by clicking the **Edit** button:

| Edit Type | Menu Option | Purpose |
|-----------|-------------|---------|
| **Edit Metadata** | Edit Metadata | Modify the entry's information fields |
| **Edit Loading Rule** | Edit Loading Rule | Modify the loading rule configuration |
| **Edit Layout** | Edit Layout | Modify the layout definition |

---

## Editing Metadata

### Step 1: Click Edit Metadata
Locate the entry you want to edit and click the **Edit Metadata** button.

### Step 2: Modify the Fields
The same dialog used for creating entries will open, but with the existing values pre-filled.

### Step 3: Save Changes
After making your changes, click **Save** to update the entry.

### What Gets Updated
- All metadata fields that you modified
- The **Modified** date is automatically updated

---

## Editing Loading Rules

For detailed instructions on editing loading rules, see the [Loading Rule Editor Guide](04_Loading_Rule_Editor.md).

### Quick Steps

1. Click **Edit** on the entry row and select **Edit Loading Rule**
2. The Loading Rule Editor opens with the current rule
3. Make your changes
4. Click **Save** to save changes to the current version

---

## Editing Layouts

For detailed instructions on editing layouts, see the [Layout Editor Guide](05_Layout_Editor.md).

### Quick Steps

1. Click **Edit Layout** on the entry row
2. The Layout Editor opens with the current layout
3. Make your changes
4. Save to update the layout

---

## Deleting an Entry

If you no longer need an entry, you can delete it.

### Step 1: Click Delete
Click the **Delete** button on the entry row.

### Step 2: Confirm Deletion
A confirmation dialog will appear. Click **Yes** to confirm or **No** to cancel.

### Important Notes

| Note | Description |
|------|-------------|
| **Permanent action** | Deleted entries cannot be recovered |
| **Files remain** | Associated loading rule and layout files are not automatically deleted |

---

## Refreshing the Table

If you have made changes outside the application or want to ensure you have the latest data, click the **Refresh** button.
