# Common Issues and Solutions

This guide provides solutions to common issues you may encounter while using the NOVA Automation Suite.

---

## NOVA Launcher Issues

### Launcher Does Not Open

| Possible Cause | Solution |
|----------------|----------|
| Application not installed properly | Reinstall the application |
| Antivirus blocking | Add the application to your antivirus exceptions |
| Missing permissions | Run as administrator |

### Automation Tool Does Not Launch

| Possible Cause | Solution |
|----------------|----------|
| Tool already running | Check if the tool is already open |
| Loading timeout | Wait longer; large tools may take time to load |
| Missing dependencies | Contact your IT team to verify installation |

---

## SDD Automation Tool Issues

### Destination Folder Selection Issues

| Issue | Possible Cause | Solution |
|-------|----------------|----------|
| Cannot select folder | Folder permissions | Choose a folder you have write access to |
| Folder not found | Path changed | Navigate to the correct location manually |
| Folder not remembered | Cache cleared | Reselect the folder; it will be remembered |

### Metadata Entry Issues

| Issue | Possible Cause | Solution |
|-------|----------------|----------|
| Cannot save entry | Required fields empty | Fill in all required fields |
| Entry disappears | Not saved properly | Ensure you click Save before closing |
| Duplicate entry error | Entry already exists | Check for existing entries with same details |

### Loading Rule Editor Issues

| Issue | Possible Cause | Solution |
|-------|----------------|----------|
| Template not loading | Wrong file format | Use the correct Excel template |
| Dropdowns empty | Layout not created | Create the layout first |
| Cannot edit cells | Read-only mode | Check if you have edit permissions |

### Layout Editor Issues

| Issue | Possible Cause | Solution |
|-------|----------------|----------|
| DSV file not uploading | Wrong format | Ensure file is Excel format with "Field Name" column |
| Text not saving | Validation error | Check for syntax errors in your layout |
| Format not matching | Wrong format selected | Select the correct format type |

### Version Control Issues

| Issue | Possible Cause | Solution |
|-------|----------------|----------|
| Cannot finalize | No version exists | Create and save a version first |
| Version not appearing | Not saved | Ensure the version was saved properly |

### SDD Consolidation Issues

| Issue | Possible Cause | Solution |
|-------|----------------|----------|
| Validation errors | Missing required fields | Fill in all required fields for each entry |
| Sheet name too long | Loading Flow name too long | Use shorter Loading Flow names |
| File not generated | Permission error | Check folder write permissions |

## Automation Suite Issues

### Module Does Not Launch

| Possible Cause | Solution |
|----------------|----------|
| Another module running | Close other running modules first |
| Window minimised | Check taskbar for minimised windows |
| Loading error | Try launching again |

### Operation Fails

| Possible Cause | Solution |
|----------------|----------|
| Invalid input file | Check the input file format and content |
| Network issue | Verify network connectivity |
| RAID portal unavailable | Check if the RAID portal is accessible |
| Missing prerequisites | Complete prerequisite operations first |

---

## RAID Automation Issues

### Browser Does Not Open

| Possible Cause | Solution |
|----------------|----------|
| Chrome not installed | Install Google Chrome |
| ChromeDriver issue | The tool manages this automatically; restart the application |
| Permissions issue | Run the application as administrator |

### Login Fails

| Possible Cause | Solution |
|----------------|----------|
| Wrong credentials | Verify username and password in RAID Config |
| Portal URL incorrect | Check the RAID Portal URL setting |
| Session expired | Re-enter credentials and try again |
| Network issue | Check network connectivity to the portal |

### Automation Stops Midway

| Possible Cause | Solution |
|----------------|----------|
| Page load timeout | Wait and try again; portal may be slow |
| Element not found | Portal interface may have changed; contact support |
| Browser crash | Close all browser windows and try again |
| Network disconnection | Restore network and try again |

---

## General Issues

### Application Freezes

| Possible Cause | Solution |
|----------------|----------|
| Large file processing | Wait for processing to complete |
| Memory issue | Close other applications to free memory |
| Background process stuck | Close and restart the application |

### Data Not Saving

| Possible Cause | Solution |
|----------------|----------|
| Disk full | Free up disk space |
| Permission denied | Check folder write permissions |
| File in use | Close other applications using the file |
| Application not responding | Wait or restart the application |

### Unexpected Error Messages

| What to Do |
|------------|
| Note the exact error message |
| Take a screenshot of the error |
| Check this guide for the specific error |
| Contact support if the issue persists |

---

## Getting Help

If the solutions above do not resolve your issue:

1. **Check the FAQ** — See [Frequently Asked Questions](03_FAQ.md)
2. **Note the details** — Write down the exact error and steps to reproduce
3. **Take screenshots** — Capture any error messages
4. **Contact support** — Reach out to the Mobileum NOVA support team
