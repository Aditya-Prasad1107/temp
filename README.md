## What is NOVA?

NOVA is an automation platform designed to simplify configuration tasks in the RAID system. It automates the creation of entities, formats, validations, and dashboards, helping to reduce manual effort and potential errors.

### Key Benefits

| Benefit | Description |
|---------|-------------|
| **Reduced Manual Effort** | Automates repetitive configuration tasks |
| **Consistency** | Applies standardised configuration patterns |
| **Error Reduction** | Reduces the chance of manual input errors |
| **Centralised Management** | Manage configurations from a single interface |

---

## What Can You Do with NOVA?

### Automation Suite
Access modules for Platform Integrity (PI) and Usage Control (UC) automation. Create entities, formats, streams, validations, and dashboards.

### SDD Automation Tool
Manage System Design Documents (SDD) with version control. Create loading rules, define layouts, and configure RAID settings.

## Getting Started

### System Requirements

Before using NOVA, ensure your computer meets these requirements:

- **Operating System**: Windows 10 or later
- **Browser**: Google Chrome (required for RAID automation features)
- **Network**: Access to your organisation's RAID portal

### Launching NOVA

1. Locate the **NOVA Automation Suite** application on your desktop or start menu
2. Double-click to launch
3. The NOVA Launcher window will appear
